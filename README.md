# OpenGL-Crystal

OpenGL 4.5 binds for Crystal 100% supported.

## Installation

1 - Add the dependency to your `shard.yml`:
```yaml
dependencies:
  src:
    github: minva/OpenGL-Crystal
```
2 - Run `shards install`
